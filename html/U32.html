NAME

Test::U32 - Designed to test the proposition that U32 is 32 bits wide

SYNOPSIS

  use Test::U32;
  check_have_u32();  # prints 1..1 ok/not ok depending
  check_u32();       # returns true is U32 is exactly 32 bits wide

DESCRIPTION

This module is no use to anyone except authors of XS extensions who want to know about the bahaviour of the U32 type defined by perl across multiple different versions of perl on different OS. The results are of this survey are kindly provided by CPANTESTERS.

EXPORT

check_have_u32()

This produces a standard 1 test TAP compliant output. See t/ for an example.

check_u32()

The actual underlying test. Returns 1 if U32 is exactly 32 bits wide.
